# AI Use Statement

AI tools were used during this project as coding assistants and learning
supports throughout the development process.

AI assistance included:
- helping interpret the structure of Google Takeout JSON files
- suggesting approaches for data cleaning and feature engineering
- assisting with debugging Python errors
- proposing visualization and analysis ideas
- explaining concepts related to text analysis and clustering

All analytical decisions, dataset selection, interpretation of results,
and written reflections were completed independently by the author.
AI-generated suggestions were reviewed, modified, and tested before
being incorporated into the final notebook.

AI was used as a support tool to accelerate learning and problem-solving,
but it did not replace understanding or independent work.

## Example AI Prompts Used During Development

- "Why are there many JSON files instead of one single file in Google Takeout exports?"
- "How can I parse and combine multiple JSON files into one dataframe using pandas?"
- "How should my GitHub repository structure look for this assignment?"
- "What visualizations would provide meaningful insights from YouTube watch history?"
- "How can I compare activity across years using my dataset?"
- "How do I create a visualization showing my top watched channels?"
- "How can I automatically categorize videos based on titles instead of manually assigning categories?"
